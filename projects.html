Meus Projetos

Ao longo da minha jornada como desenvolvedor, desenvolvi projetos voltados para diferentes áreas do desenvolvimento backend, desde APIs REST e sistemas de gestão até aplicações com processamento em lote, cache distribuído e deploy em nuvem. Cada projeto representa um novo desafio técnico e a aplicação de boas práticas de engenharia de software.

Projetos em destaque

DSCommerce API

API RESTful robusta para e-commerce com controle de acesso baseado em perfis (RBAC). Possui catálogo paginado, busca dinâmica, registro de pedidos e fluxo de segurança avançado.

  • Java 21
  • Spring Boot 3
  • Spring Security
  • OAuth2 & JWT
  • PostgreSQL
  • Hibernate
GitHub

FilePack API

API de alta disponibilidade implantada na AWS ECS Fargate via CloudFormation (IaC). Processa e criptografa múltiplos arquivos em pacotes ZIP via AES, monitorada em tempo real com CloudWatch.

  • Java 25
  • Spring Boot
  • AWS ECS Fargate
  • CloudFormation
  • CloudWatch
  • Docker
GitHub

Fipe Search POC

Prova de Conceito (POC) de alto desempenho focada em mitigar Cache Stampede e otimizar queries distribuídas usando Redis em memória. Validada com testes de estresse distribuídos via Grafana k6.

  • Java 25
  • Spring Boot 4
  • Redis
  • PostgreSQL
  • Grafana k6
  • Docker
GitHub
